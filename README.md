# 📦 My Inventory App

一個簡易的 **前後端分離庫存管理系統**  
前端使用 **Vue 3 + Vite + Element Plus**  
後端使用 **Node.js + Express + SQLite**

---

## 📁 專案結構

```
my-inventory-app/
├─ backend/ # Node.js + Express API
├─ frontend/ # Vue 3 + Vite 前端專案
├─ node_modules/
├─ package.json # 一鍵啟動前後端
├─ package-lock.json
├─ uploads/ # 圖片上傳目錄 (自動產生)
```

---

## ✅ 已完成功能

- ✔️ 產品庫存 CRUD（新增 / 修改 / 刪除 / 列表）
- ✔️ 分頁顯示
- ✔️ 即時搜尋
- ✔️ 自動圖片上傳與預覽
- ✔️ 前後端同時啟動 (`npm-run-all`)
